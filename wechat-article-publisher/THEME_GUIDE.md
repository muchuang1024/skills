# doocs/md 主题选择指南

## 可用主题列表

根据 doocs/md 排版工具，以下是可用的主题：

### 1. **orange** - 橙心主题
- **适用场景**：温暖、励志、鸡汤类文章
- **色调**：橙色、金色
- **风格**：温暖、有活力
- **示例**：女性励志、个人成长、生活感悟

### 2. **purple** - 紫色主题
- **适用场景**：优雅、高端、品质类文章
- **色调**：紫色、深紫
- **风格**：优雅、神秘
- **示例**：品牌故事、高端产品、艺术文化

### 3. **green** - 绿色主题
- **适用场景**：健康、环保、自然类文章
- **色调**：绿色、清新
- **风格**：清新、自然
- **示例**：健康养生、环保公益、户外旅行

### 4. **blue** - 蓝色主题
- **适用场景**：科技、商务、专业类文章
- **色调**：蓝色、深蓝
- **风格**：专业、理性
- **示例**：科技资讯、商业分析、职场干货

### 5. **red** - 红色主题
- **适用场景**：热情、节日、重要通知类文章
- **色调**：红色、暖色
- **风格**：热情、醒目
- **示例**：节日祝福、重要公告、活动宣传

### 6. **cyan** - 青色主题
- **适用场景**：清爽、文艺、小清新类文章
- **色调**：青色、浅蓝
- **风格**：清爽、文艺
- **示例**：文艺随笔、旅行游记、生活美学

### 7. **black** - 黑色主题
- **适用场景**：高端、简约、极简类文章
- **色调**：黑色、灰色
- **风格**：简约、高级
- **示例**：设计作品、摄影作品、极简生活

### 8. **pink** - 粉色主题
- **适用场景**：少女、甜美、浪漫类文章
- **色调**：粉色、浅粉
- **风格**：甜美、浪漫
- **示例**：美妆时尚、情感故事、少女心内容

## 主题选择原则

### 根据文章类型选择

| 文章类型 | 推荐主题 | 理由 |
|---------|---------|------|
| 鸡汤励志 | orange | 温暖有活力，符合励志氛围 |
| 职场干货 | blue | 专业理性，符合商务风格 |
| 健康养生 | green | 清新自然，符合健康主题 |
| 科技资讯 | blue/cyan | 专业或清爽，符合科技感 |
| 情感故事 | pink/purple | 浪漫或优雅，符合情感基调 |
| 品牌故事 | purple/black | 高端优雅，符合品牌调性 |
| 节日祝福 | red | 热情醒目，符合节日氛围 |
| 旅行游记 | cyan/green | 清爽自然，符合旅行感觉 |

### 根据文章情绪选择

- **积极向上、励志** → orange（橙色）
- **专业严谨、理性** → blue（蓝色）
- **清新自然、放松** → green（绿色）或 cyan（青色）
- **优雅高端、品质** → purple（紫色）或 black（黑色）
- **热情活力、醒目** → red（红色）
- **浪漫甜美、温柔** → pink（粉色）

### 根据目标受众选择

- **女性受众** → orange, pink, purple
- **男性受众** → blue, black, cyan
- **年轻人** → orange, pink, cyan
- **商务人士** → blue, black, purple
- **家庭主妇** → green, orange, pink

## 使用方法

### 在代码中指定主题

```bash
python3 scripts/markdown_to_wechat_doocs.py \
  --input article.md \
  --output article.html \
  --theme orange  # 根据文章内容选择合适的主题
```

### Agent 自动选择主题

Agent 会根据以下因素自动选择主题：

1. **文章标题**：分析标题关键词
2. **文章内容**：分析正文主题和情绪
3. **目标受众**：根据用户指定的受众群体
4. **文章类型**：鸡汤、干货、资讯等

**示例**：
- 标题："当女人一门心思搞钱，她就开始长脑子" → **orange**（励志、女性、温暖）
- 标题："2024年AI技术发展趋势分析" → **blue**（科技、专业、理性）
- 标题："春天来了，去这10个地方旅行吧" → **cyan**（旅行、清爽、文艺）

## 注意事项

1. **不要固定主题**：每篇文章根据内容选择合适的主题
2. **考虑受众**：不同受众对颜色的偏好不同
3. **保持一致**：同一系列文章可以使用相同主题
4. **测试效果**：可以先预览效果再决定

## 配置说明

在 `.env` 文件中：
```bash
# MARKDOWN_THEME 不设置默认值
# 由 Agent 根据文章内容动态选择
MARKDOWN_CONVERTER=doocs
```

在代码中使用：
```python
from config import get_markdown_config

# 获取配置
config = get_markdown_config()
converter = config['converter']  # 'doocs'
theme = config.get('theme', None)  # None（由 Agent 选择）

# Agent 根据文章内容选择主题
if "励志" in article_title or "女性" in article_title:
    theme = "orange"
elif "科技" in article_title or "商务" in article_title:
    theme = "blue"
elif "健康" in article_title or "养生" in article_title:
    theme = "green"
# ... 更多规则
```
